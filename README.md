# Learn Any Concept

AI-powered learning app for Dashtop. Type any concept and get a clear explanation at your level.

## Features

- **Explain anything** — Type any topic, get an instant explanation
- **Three levels** — Beginner, Intermediate, Expert
- **Three styles** — ELI5, Textbook, Analogy
- **Quick topics** — One-click popular concepts
- **Go deeper** — Ask for more detail on any explanation
- **Quiz mode** — Test your understanding

## Install

Install in Dashtop via Settings → Packages, or:

```bash
npm install
npm run pack
# → dist/dashtop-ai-learn-any-concept-v1.0.0.dashtop
```

## Develop

```bash
npm install
npm run dev
```
